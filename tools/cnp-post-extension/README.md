# CNP TIMES 投稿ツール（Chrome拡張）

ひろゆきさんへ。Discordに書いた分析コメントを、選択→右クリック→確認→保存の
最小手順でCNP TIMESに掲載できるようにするツールです。ストアには公開していないので、
少しだけ手動でのセットアップが必要です。以下の3ステップだけで使えるようになります。

## セットアップ（最初の1回だけ）

### ステップ1: フォルダを用意する

このツール一式（`cnp-post-extension` フォルダ）を、パソコンの分かりやすい場所に置いてください
（デスクトップなど。zipで届いた場合は解凍してからのフォルダを使います）。
このフォルダ自体は消さずにそのまま残しておいてください（拡張機能がここから読み込まれ続けます）。

### ステップ2: Chromeに読み込む

1. Chromeのアドレスバーに `chrome://extensions` と入力して開く
2. 画面右上の「デベロッパーモード」のスイッチをONにする
3. 左上に出てくる「パッケージ化されていない拡張機能を読み込む」ボタンをクリック
4. ステップ1で用意した `cnp-post-extension` フォルダを選択する
5. 一覧に「CNP TIMES 投稿ツール」が表示されればOKです

Chromeのツールバー（アドレスバーの右側）にアイコンが表示されない場合は、
パズルのピース🧩のアイコンをクリックして「CNP TIMES 投稿ツール」をピン留めしておくと
次から見つけやすくなります。

### ステップ3: APIキーを設定する

1. 拡張機能一覧の「CNP TIMES 投稿ツール」の「詳細」→「拡張機能のオプション」を開く
   （または、ツールバーのアイコンを右クリックして「オプション」を選ぶ）
2. るくさんから渡されたAPIキー（`cnpt_` から始まる文字列）を貼り付ける
3. 「接続テスト」を押して「✅ 接続OK（○件の記事が見えています）」と出ることを確認する
4. 「保存」を押す

これでセットアップは完了です。APIキーはこのパソコンのChromeの中だけに保存され、
どこにも送信されません。

## 使い方

### 方法1: 選択して右クリック（おすすめ）

1. Discordの投稿画面（ブラウザ版）で、投稿した分析コメントの文章を選択（ドラッグ）する
2. 選択した部分を右クリックし、「CNP TIMESに投稿」を選ぶ
3. 小さな画面（フォーム）が開き、選択した文章が「本文」欄に自動で入っています
4. 内容を確認・編集して「保存」を押せば完了です

### 方法2: ツールバーのアイコンから開く

デスクトップ版Discordなど、右クリックメニューが使えない場所からコピーした文章を貼りたい場合は、
ツールバーのアイコンをクリックしてください。空のフォームが開くので、本文欄に貼り付けてから
保存してください。

### フォームの項目について

- **日付**: 掲載日です。最初から「昨日の日付」が自動で入っています
  （分析コメントを書いた翌日に掲載する運用のため）。別の日に掲載したい場合はここを変更してください。
- **タイトル**: 本文から自動で候補が入ります（「分析1502回目」のような行や【】の見出しを優先的に拾います）。
  気に入らなければ自由に書き換えてください。
- **本文**: Markdown形式で入力します。見出しは書かなくてもそのまま文章として表示されます。
- **画像を貼りたいとき**: 本文欄をクリックしてカーソルを合わせた状態で、
  スクリーンショットなどの画像をコピーしてから `Cmd+V`（Windowsなら `Ctrl+V`）でペーストしてください。
  自動でアップロードされ、カーソル位置に画像が挿入されます。

### 既に記事がある日付を選んだ場合

日付を変更したときに、その日付に既に記事がある場合は
「⚠ この日付には既に記事があります。保存すると上書きされます」という警告が出ます。
うっかり上書きしないよう注意してください。今書いている内容ではなく、
既存の記事を読み込んで直したい場合は「既存内容を読み込む」ボタンを押すと、
既存のタイトル・本文がフォームに読み込まれます。

## うまくいかないとき

- **「APIキーが設定されていません」と出る**: オプション画面でAPIキーを保存し直してください。
- **保存しようとすると「APIキーを確認してください」と出る**: APIキーが間違っているか、
  無効化されている可能性があります。るくさんに確認してください。
- **「通信エラーが発生しました」と出る**: インターネット接続を確認して、少し時間をおいてから
  もう一度保存を試してください。
- それでも解決しない場合は、るくさんに画面のスクリーンショットと一緒に連絡してください。

## セキュリティについて

- APIキーはこのパソコンのChromeの中だけに保存されます。他の人と共有したり、
  メールやチャットにそのまま貼り付けたりしないでください。
- もしAPIキーが漏れてしまった、または心当たりのない投稿がされているなどがあれば、
  すぐにるくさんに連絡してください（キーを無効化して再発行できます）。
